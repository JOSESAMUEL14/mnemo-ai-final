Mnemo AI — Developer API

Build on top of Mnemo's memory layer

Endpoints
POST /api/remember
Saves a new memory entry into Mnemo's memory layer, optionally scoped to a dataset.
Request
{ "entry": "string", "dataset": "string (optional)" }
Response
{ "status": "success", "message": "Memory saved!" }
POST /api/recall
Queries stored memories and returns relevant matches.
Request
{ "query": "string" }
Response
{ "status": "success", "results": ["string", ...] }
POST /api/forget
Permanently deletes memories, optionally scoped to a single dataset instead of everything.
Request
{ "dataset": "string (optional)" }
Response
{ "status": "success", "message": "Memory forgotten!" }
GET /api/stats
Returns current usage stats — memory count, active days, goals and growth score.
Response
{ "memories": int, "days_active": int, "goals": int, "growth_score": int }
GET /api/insights
Returns computed life-intelligence scores, a behavioral "DNA" breakdown, and future predictions.
Response
{ "score": int, "resilience": int, "growth": int, "consistency": int, "positivity": int, "dna": {...}, "predictions": [...] }
